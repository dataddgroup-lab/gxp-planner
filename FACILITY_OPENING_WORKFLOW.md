## PHASE 1: DESIGN & LAYOUT (Weeks 5–8)

### Step 4: Facility Layout Design
**Objective:** Create a GMP-compliant facility layout

**What GxP Planner Does:**
- **2D Layout Tool:** Drag equipment, rooms, flows onto canvas
- **Compliance Checking:** Real-time validation (ISO 14644, contamination prevention, unidirectional flow)
- **Standards Enforcement:**
  - Cleanroom classification (ISO 5/7/8 per product type)
  - Pressure cascades (positive differentials enforced)

**Regulatory Checkpoints:**
- ✅ Layout complies with GMP (21 CFR 211.22 / EMA Annex 1)
- ✅ Cleanroom classification confirmed (ISO 14644)
- ✅ HVAC zones defined (separate supply/exhaust per area)
- ✅ Environmental monitoring points identified
- ✅ Utility segregation confirmed (no cross-contamination risk)
- ✅ Personnel flow prevents contamination

**In-App Guidance:**
> "Your facility layout is the foundation. Get it wrong, and validation is impossible. GxP Planner's layout tool prevents contamination risks by design. Red X = non-compliant. Green ✓ = approval-ready."

## IN-APP ASSISTANT BEHAVIOR

**The in-app assistant (powered by the same Agent Framework as the build process) guides customers:**

### Guidance Principles
1. **Clarity:** No jargon, explain regulations plainly
2. **Safety rails:** Never let customers skip required steps
3. **Proactive:** Alert when milestones approach, deadlines loom
4. **Encouraging:** Celebrate progress, recognize complexity

### Example Interactions

**Customer:** "I'm ready to order equipment."
**Assistant:** "Great! Before ordering, let me verify:
- ✅ Facility layout approved? 
- ✅ Equipment specifications locked?
- ✅ Lead times confirmed (critical path)?
- ✅ Supplier quality certifications verified?
If all yes, equipment order will unlock. If any no, let's fix that first."

**Customer:** "Can I skip IQ and go straight to OQ?"
**Assistant:** "No. IQ (Installation Qualification) proves equipment was installed correctly. OQ tests if it operates correctly. You can't test if you haven't installed. We're enforcing IQ-before-OQ in the system."

**Customer:** "We found a defect in a batch."
**Assistant:** "Found a defect = you have a deviation. GxP Planner just unlocked the deviation tracker:
1. Describe the defect (what, when, which batch?)
2. Assess impact (does it affect product safety/efficacy?)
3. Investigate root cause (why did this happen?)
4. Implement CAPA (corrective + preventive actions)
5. Verify closure (prove the fix works)
We'll guide each step. No shortcuts."
